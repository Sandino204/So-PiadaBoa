# Só Piada Boa API

## Description
- Uma api que permite ler, interagir e criar piadas
- o usuario pode criar um usuario e atravez dele criar pidas e interagir com as piadas existentes

## Commands

### Test the API
```
    npm run test
```

### Run the API
```
    docker compose up
```

## Documentation
